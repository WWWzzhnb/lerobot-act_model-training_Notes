### 简介
本文档聚焦于**RTX 5090 GPU 硬件环境下，LeRobot 0.4.4 框架的部署及 ACT 模型训练全流程**，针对 LeRobot 0.4.4 默认依赖与 RTX 5090 兼容性冲突问题，提供了从服务器连接、数据集校验、环境适配（含 PyTorch、FFmpeg、torchcodec 等核心组件版本调优），到 ACT 模型训练、GPU 状态监控的完整解决方案。

文档核心解决了 LeRobot 依赖版本限制、RTX 5090 驱动/CUDA 适配、FFmpeg 库路径配置等关键问题，通过修改 LeRobot 依赖配置文件、指定适配版本的 PyTorch 安装、配置系统库路径等操作，确保环境兼容性；同时给出标准化的 ACT 模型训练命令与 GPU 监控方法，可直接指导基于指定服务器的数据集的 ACT 模型训练落地，适用于机器人学习领域基于 RTX 5090 的 LeRobot 框架实践。
